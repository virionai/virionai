## The thesis

The durable output of AI work should not be a chat transcript.

It should be a portable artifact another person, model, system, auditor, or regulator can open later, verify offline, and continue without trusting the original runtime.

Virion builds the instruments underneath that shift: evidence substrates, portable work products, model harnesses, human workspaces, and evaluation loops.

## Capsules

Capsule is an open protocol for portable AI work artifacts that carry content, state, provenance, participants, and a verifiable event ledger in a single `.capsule` file.

## Open work

- [capsules-protocol](https://github.com/virionai/capsules-protocol): open protocol, SDKs, CLI, verifier, examples, and conformance harness.
- [operators](https://github.com/virionai/operators): local-first workspace for evidence, Gemma-assisted reasoning, ledger events, and Capsule handoff.
- [capsules-gemma-4-good](https://github.com/virionai/capsules-gemma-4-good): Gemma 4 Good work around portable, verifiable Capsule artifacts.
