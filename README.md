<h1 align="center">Virion</h1>

<p align="center">
  Intelligence-to-intelligence systems for portable, verifiable AI work.
</p>

## The thesis

The durable output of AI work should not be a chat transcript.

It should be a portable artifact another person, model, system, auditor, or regulator can open later, verify offline, and continue without trusting the original runtime.

Virion builds the instruments underneath that shift: evidence substrates, portable work products, model harnesses, human workspaces, and evaluation loops.

## Explore the code

Three open repositories carry the work. Start here.

### [capsules-protocol](https://github.com/virionai/capsules-protocol)

![Rust](https://img.shields.io/badge/Rust-0A0D0F?style=flat-square&logo=rust&logoColor=C9A87B)
![JavaScript](https://img.shields.io/badge/JavaScript-0A0D0F?style=flat-square&logo=javascript&logoColor=C9A87B)
![Python](https://img.shields.io/badge/Python-0A0D0F?style=flat-square&logo=python&logoColor=C9A87B)
![License: MIT](https://img.shields.io/badge/license-MIT-C9A87B?style=flat-square)

The canonical open protocol for portable, signed, AI-readable records of multi-actor work — spec, SDKs, CLI, verifier, and conformance harness. Clone it and verify a capsule offline in one command.

### [operators](https://github.com/virionai/operators)

![TypeScript](https://img.shields.io/badge/TypeScript-0A0D0F?style=flat-square&logo=typescript&logoColor=7BA7C9)

An offline, local-first investigative workspace — local Gemma via Ollama, a malleable evidence canvas, visible ledger events, and Capsule handoff. No cloud model required.

### [researcher-brain](https://github.com/virionai/researcher-brain)

![Logseq](https://img.shields.io/badge/Logseq-0A0D0F?style=flat-square&logo=logseq&logoColor=7BA7C9)
![Method: Karpathy Loop](https://img.shields.io/badge/method-Karpathy%20Loop-C9A87B?style=flat-square)

A second brain for research, built for the era when AI doesn't just retrieve knowledge — it produces it. Structures a corpus so the propose → run → measure → commit loop runs on your own knowledge.
